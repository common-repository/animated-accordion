Accordion Animation allows the user to add nice accordion,with text and image and control the layout in easy way.

== Description ==

Accordion Animation is light weight plugin and it seo friendly and depends on CSS3 and jQuery,it has a really simple
backend to make it easier for newbies,it's not new idea but it's really helps you in your content,and it's a 
responsive plugin with its layouts.



== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/animated_Accordion` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Dashboard Name screen to configure the plugin
